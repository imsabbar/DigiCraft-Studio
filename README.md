# DigiCraft Studio

DigiCraft Studio is a modern, responsive landing page template designed for digital agencies, creative studios, and freelance portfolios. It features a clean design, smooth animations, and a fully functional layout for showcasing services and projects.

- **Responsive Design**: Fully responsive layout that adapts to all screen sizes (Desktop, Tablet, Mobile).
- **Modern UI/UX**: Clean aesthetic with custom CSS variables for easy theming.
- **Interactive Elements**:
  - Mobile-friendly navigation menu with toggle.
  - Sticky header with scroll effect.
  - Portfolio filtering functionality.
  - Scroll reveal animations for sections.
  - "Back to Top" button.
- **Sections**:
  - **Hero**: Engaging landing section with call-to-action.
  - **Services**: Grid layout highlighting agency services.
  - **Features**: Key selling points and agency strengths.
  - **Portfolio**: Filterable gallery of projects (Web Design, Apps, Branding, etc.).
  - **Newsletter**: Subscription form section.
  - **Contact**: Detailed contact form and information.
  - **Footer**: Social links, useful links, and Instagram feed widget.
